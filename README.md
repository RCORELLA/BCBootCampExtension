# BCBootCamp Customer Classification

This App has been created to train new AL Developers

## 📋 Description

This Business Central extension provides a comprehensive customer classification system based on sales performance and customer quality metrics. It enables businesses to segment and categorize their customer base for better analysis, targeted marketing, and strategic decision-making.

## ✨ Features

- **Flexible Categorization**: Customizable classification rules to fit your business needs
- **Enhanced Customer Cards**: Extended customer information with classification data


## 🎯 Business Benefits

- Identify high-value customers for priority service
- Target marketing campaigns to specific customer segments
- Optimize sales strategies based on customer classification
- Improve customer relationship management
- Make data-driven decisions about customer engagement

## 📦 Requirements

- **Business Central Version**: 27.0 or higher
- **Platform**: Cloud 
- **Dependencies**: Base Application



## 📊 Usage

### Viewing Customer Classification

- Open any **Customer Card**
- View the **Classification** field showing the customer category


### Running Classification Updates

- Go to **Customer Classification** page
- Select **Actions** → **Run Classification**
